# 🏀 NBA Player Efficiency & Value Dashboard (2025–26 Season)

Excited to share my latest NBA Player Efficiency & Value Dashboard, built to analyze player performance, roster depth, and hidden talent across the league using data analytics and visualization techniques.

🔍 Dashboard Highlights

✅ Top 20 Players by Value Score

Identifies the league's most impactful players based on a custom weighted value metric.
Features elite performers such as Nikola Jokić, Victor Wembanyama, and Giannis Antetokounmpo.

✅ Efficiency vs. Scoring Analysis

Visualizes the relationship between total points scored and overall efficiency.
Highlights players who generate exceptional impact beyond scoring alone.

✅ Team Roster Depth Breakdown

Categorizes players into Stars, Starters, Rotation Players, and Bench contributors.
Provides insights into team construction and depth across the league.

✅ Hidden Gems Detection

Identifies high-value players producing efficiently in limited minutes (15–28 MPG).
Useful for discovering breakout candidates and underrated contributors.
